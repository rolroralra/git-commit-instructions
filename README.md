> .github 디렉터리에 git-commit-instructions.md 파일을 설정하는 것은 주로 GitHub Copilot과 같은 AI 도구에 커밋 메시지 작성 지침을 제공하는 데 사용됩니다. 이 파일은 팀원들이 참고할 수 있는 커밋 컨벤션 문서로도 활용될 수 있습니다. 

# 설정 방법
- 리포지토리 루트에 .github 디렉터리 생성: 로컬 리포지토리의 최상위 경로에 .github 폴더가 없다면 생성합니다.

```bash
mkdir -p .github
```

- git-commit-instructions.md 파일 생성: .github 디렉터리 내에 git-commit-instructions.md 파일을 생성합니다.
```bash
touch .github/git-commit-instructions.md
```
